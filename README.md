## Overview
Repositorio de bases de datos sobre conectómica cerebral humana: conectividad estructural (DTI, tractografía) y funcional (fMRI), incluyendo mapas de redes cerebrales y variabilidad interindividual. Datos compilados y organizados para investigación en neurociencias. Incluye: nombre, acrónimo, URL, año, institución, tipo de datos, número de sujetos, características principales, herramientas y refer
